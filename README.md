# Morning Brief — Ops Dashboard

Private operational dashboard for Ryan Graves.

## Usage

Open `index.html` in any browser. No build tools or server required.

## Editing Data

All dashboard content lives in `data.js`. Edit priorities, entity details, and contacts there.

## Structure

- `index.html` — Page markup
- `style.css` — Dark-mode cockpit styling
- `app.js` — Rendering logic and interactivity
- `data.js` — All editable dashboard data
